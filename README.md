# Flow State - Weekly, Monthly Productivity & Focus App

Flow State is a comprehensive productivity tool built with Flutter, designed to help users organize their life through structured planning and focus techniques. The app provides a multi-layered approach to task management, combining short-term and long-term planning with focus-driven tools.

## Key Features

* **Multi-Level Planning:** Organized systems for Weekly, Monthly, and Daily planning to track tasks across different timeframes.
* **Deadlines & Tracking:** Set specific deadlines for tasks to ensure accountability and timely completion.
* **Focus Timer:** Integrated timer functionality to help users enter "Flow State" and maintain deep work sessions.
* **Daily Streaks:** A habit-tracking system that rewards consistency through streaks, encouraging users to maintain their productivity momentum.
* **Smart Notification System:** Automated reminders for task start times, deadlines, and daily plan completions.
* **State Management:** Efficiently managed using the Provider pattern for a smooth and responsive user experience.
* **Modern UI:** Distraction-free interface following Material Design 3, optimized for productivity.

## Technical Stack

* Framework: Flutter
* Programming Language: Dart
* State Management: Provider
* Local Notifications: flutter_local_notifications
* Persistence: Shared Preferences

## Installation and Setup

1. Ensure Flutter SDK is installed.
2. Clone the repository:
   git clone https://github.com/omarsuliman1/Flow_State.git
3. Navigate to the project directory:
   cd Flow_State
4. Install dependencies:
   flutter pub get
5. Run the application:
   flutter run

## Project Structure

* lib/providers: Logic for planning (Weekly/Monthly), Streaks, and Timer state.
* lib/services: Notification services and background persistence logic.
* lib/models: Data models for tasks, deadlines, and user progress.
* lib/screens: UI views for the dashboard, timers, and planning modules.
